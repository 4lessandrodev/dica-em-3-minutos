# Variáveis

## Variáveis em javascript
Existem três tipos de declarações em JavaScript.

- var
- let
- const

|              |      var       |      const     |      let       |
|--------------|:--------------:|---------------:|---------------:|
| escopo       | local & global | bloco          | bloco          |
| redeclarável | sim            | não            | não            |
| reatribuível | sim            | não            | sim            |
| içável       | sim            | não            | não            |
